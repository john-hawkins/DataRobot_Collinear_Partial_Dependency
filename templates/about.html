Collinear Partial Dependency Plot

Use this application to generate a custom partial dependency plots for a variable that has a collinear relationship with other variables in your model.

Once generated, the plots will be saved in the folder /static and reused if you request them again. The file name will be as follows: 

   PRJECT_ID-MODEL_ID-FOCUS_COLNAME-COLONE_NAME-COLTWO_NAME.png

If you want to regenerate an existing plot (for the same variable and model) using an alternative dataset you will need to delete the old version.
